# The-SAT-Study-Guide
The SAT Study Guide is a tool designed to help students with the SAT by providing a clean interface that they can use to study the testable formulas that may be given on the actual SAT. Embedded is the official graphing calculator that will be used on test day as well as formulas and examples to help students with the test.
